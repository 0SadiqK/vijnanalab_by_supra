# VijnanaLab by Supra

An interactive web project built with Vite + TypeScript.

## Overview
VijnanaLab is a modular frontend setup meant for scalable experiments and projects. It includes pages, components, and services structured for clean development.
---
## Features
- Modular components
- Fast dev with Vite
- TypeScript everywhere
- API handlers via services
- Ready for expansion
---
## Tech Stack
- TypeScript
- React
- Vite
- HTML/CSS
---
## Installation
```bash
git clone https://github.com/0SadiqK/vijnanalab_by_supra.git
cd vijnanalab_by_supra
npm install
```
---

## Environment Variables

Create .env.local and add:

```bash
GEMINI_API_KEY=your_api_key_here
```
---

## Run Dev Server
```bash
npm run dev
```
---
## Project Structure
- components/        UI components
- pages/             Views & routes
- services/          API & helpers
- App.tsx            App entry
- index.tsx          Render
- vite.config.ts     Config
